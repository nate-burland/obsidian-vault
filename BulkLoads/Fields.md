| Readable Field Name    | Description                                                                    | Options                                                                                                                                                                                                                                                                                                            |
| ---------------------- | ------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Load Number            | Unique identifier assigned to the shipment or load for tracking and reference. |                                                                                                                                                                                                                                                                                                                    |
| Origin City            | City where the shipment originated.                                            |                                                                                                                                                                                                                                                                                                                    |
| Origin State           | State abbreviation where the shipment originated.                              |                                                                                                                                                                                                                                                                                                                    |
| Destination City       | City where the shipment is to be delivered.                                    |                                                                                                                                                                                                                                                                                                                    |
| Destination State      | State abbreviation where the shipment is to be delivered.                      |                                                                                                                                                                                                                                                                                                                    |
| Driver Name            | Full name of the driver transporting the shipment.                             |                                                                                                                                                                                                                                                                                                                    |
| Company Name           | Name of the trucking or logistics company handling the shipment.               |                                                                                                                                                                                                                                                                                                                    |
| Date                   | Date printed on the document.                                                  |                                                                                                                                                                                                                                                                                                                    |
| Time                   | Time printed on the document.                                                  |                                                                                                                                                                                                                                                                                                                    |
| Commodity              | Type of goods or material being shipped.                                       |                                                                                                                                                                                                                                                                                                                    |
| Net Weight             | Total net weight of the shipment, typically in pounds; net = gross – tare.     | Example: 48,100 lbs                                                                                                                                                                                                                                                                                                |
| Rate Amount            | Rate charged per unit of weight or commodity.                                  | Example: 6.00                                                                                                                                                                                                                                                                                                      |
| Rate Unit              | Unit of measure for the rate.                                                  | Ton, Bushel, Flat, Pound, etc.                                                                                                                                                                                                                                                                                     |
| Booked Through Company | Company or broker through which the load was booked.                           |                                                                                                                                                                                                                                                                                                                    |
| Number of Loads        | Total number of loads represented on the document.                             |                                                                                                                                                                                                                                                                                                                    |
| Document Category      | Type of document.                                                              | Funding_Sheet – Smart Freight Load Funding Sheet  <br>Rate_Confirmation – Broker/company rate confirmation  <br>Scale_Ticket – Commodity scale ticket  <br>Bill_Of_Lading – Bill of Lading  <br>Load_Invoice – Invoice from trucking/logistics company  <br>Surcharge_Receipt – Document listing surcharge details |
| Surcharges             | List of surcharge items for the load.                                          | Each surcharge includes:  <br>- Surcharge Type (Fuel, Wait Time, Demurrage)  <br>- Surcharge Value (Dollar amount)                                                                                                                                                                                                 |
| Ticket Number          | Unique number usually printed on a scale ticket.                               |                                                                                                                                                                                                                                                                                                                    |
